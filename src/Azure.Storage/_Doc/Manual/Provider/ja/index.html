BlobStorageController

BlobStorageControllerは、Azure Blob Storageへの接続とファイル操作を行うためのコントローラです。
JSONオプションに指定する項目を以下に示します。各項目はJSON記法に則ってKeyと対応するValueを入力してください。

型名

GeoLibrary.ORiN3.Provider.Azure.Storage.O3Object.Controller.BlobStorageController, GeoLibrary.ORiN3.Provider.Azure.Storage

JSONオプション

Key 型 必須 暗号化 説明 値範囲 デフォルト値
Account Name string ✓ 接続するAzure Storageアカウント名です。 ストレージアカウント名の命名規則に準拠
Account Key string ✓ ✓ Azure Storageアカウントのアクセスキーです。 有効なアクセスキー
Use Https bool HTTPSを使用するかどうか `true` or `false` true
Endpoint Suffix string Azureのエンドポイントサフィックス 任意のサフィックス core.windows.net
Proxy Uri string プロキシサーバーのアドレス 有効なURI

UploadBlockFromFile

Blobの名前はローカルのファイル名と同じになります。
ローカルのディレクトリ名は使用しないので必要に応じてPrefixを指定してください。

Overwrite bool 既存Blobを上書きするかどうか true

戻り値

Key 型 説明

UploadBlockFromDirectory

ローカルディレクトリ内の全ファイルをAzure Blob StorageにBlock Blobとしてアップロードします。
サブディレクトリ以下にあるファイルはアップロード対象外です。
処理の途中でエラーが発生した場合、以降のファイルアップロードは中断します。
アップロードに成功したファイルの数は戻り値の『Uploaded Count』で確認できます。
Blobの名前はローカルのファイル名と同じになります。
ローカルのディレクトリ名は使用しないので必要に応じてPrefixを指定してください。

引数

Key 型 必須 説明 デフォルト値
Directory Path string ✓ アップロード対象ディレクトリのパス
Container Name string ✓ アップロード先コンテナ名
Prefix string Blob名の接頭辞
Overwrite bool 既存Blobを上書きするか true

戻り値

Key 型 説明
Result int 実行結果コード
Uploaded Count int アップロードしたファイル数
Azure Error Code string Azureエラーコード
HTTP Status int HTTPステータスコード
Error Message string エラーメッセージ
Stack Trace string スタックトレース

BlockBlobFile

BlockBlobFileは、Azure Blob StorageのBlock Blobを扱うためのクラスです。
主に以下の操作を行います：

  • Azure Storageアカウント名とアクセスキーを使って接続
  • 指定されたBlobファイルをダウンロードしてローカルに保持
  • 取得したBlobファイルを読み取ることができます
このオブジェクトは読み取り専用でデータを書き込むことはできません。
JSONオプションに指定する項目を以下に示します。各項目はJSON記法に則ってKeyと対応するValueを入力してください。

型名

GeoLibrary.ORiN3.Provider.Azure.Storage.O3Object.File.BlockBlobFile, GeoLibrary.ORiN3.Provider.Azure.Storage

JSONオプション

Key 型 必須 説明 値範囲 デフォルト値
Container Name string ✓ 操作対象のコンテナ名 任意のコンテナ名
Blob Path string ✓ 対象ファイルのパス 任意のパス

PageBlobFile

PageBlobFileは、Azure Blob StorageのPage Blobを扱うためのクラスです。
主に以下の操作を行います：

  • Azure Storageアカウント名とアクセスキーを使って接続
  • 指定されたBlobファイルに直接アクセスして読書き
必要に応じて新規にPage Blobを生成することもできますが、Page Blobのサイズは512の倍数にする必要があります。
Lengthはファイル生成時にしか使いません。
JSONオプションに指定する項目を以下に示します。各項目はJSON記法に則ってKeyと対応するValueを入力してください。

型名

GeoLibrary.ORiN3.Provider.Azure.Storage.O3Object.File.PageBlobFile, GeoLibrary.ORiN3.Provider.Azure.Storage

JSONオプション

Key 型 必須 説明 値範囲 デフォルト値
Container Name string ✓ 操作対象のコンテナ名 任意のコンテナ名
Blob Path string ✓ 対象ファイルのパス 任意のパス
Create New If Not Exists bool ファイル生成フラグ
Length long ファイルサイズ 正の値

Appendix

Result

  • Unknown (-1): Unknown error
  • Success (0): Operation completed successfully
  • AzureError (1): Error reported from Azure SDK or service
  • OtherError (2): Other errors (network issues, unexpected errors)
