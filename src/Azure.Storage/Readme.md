## GeoLibrary ORiN3 Provider Azure Storage

`GeoLibrary.ORiN3.Provider.Azure.Storage` is a ORiN3 provider that writes/reads data to/from Azure Storage.

### Supported Devices
These storage are supported.
- Blob Storage
  - Block Blob
  - Append Blob
  - Page Blob
- File Storage

---
